# Версии импорта и сравнение

Каждый импорт Excel сохраняется как отдельная версия данных.  
Версия определяется `import_run_id`.

## Принципы

- Импорт не затирает прошлые версии.
- Ручные записи (`import_run_id = NULL`) всегда доступны.
- Отчёты поддерживают параметр `import_run_id` (если не передан — берётся последняя успешная версия).

## Сравнение версий

Эндпойнт:

```
GET /imports/compare
```

Параметры:
- `project_id`
- `run_a`, `run_b`
- `date_from`, `date_to`
- `by` (`wbs|discipline|block|floor|ugpr`)
- `scenario` (`plan|forecast|actual`)

В UI: вкладка **Импорт** → блок «Сравнение версий».
