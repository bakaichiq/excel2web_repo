# Формат Excel для импорта

Ниже описаны ключевые листы и ожидания по колонкам.  
Парсеры живут в `backend/app/services/etl/parsers/`.

## 1) ВДЦ

Лист: `ВДЦ`  
Ключевые колонки:
- `Идентификатор операции`
- `Категория`
- `Наименование работ и материалов`
- `Ед. изм`
- `Количество Защита`
- `Цена Защита`
- `Цена Фактическая` (опционально)

Датные колонки (дни) распаковываются в факт по дням.

## 2) ГПР

Лист: `ГПР`  
Ключевые колонки:
- `Идентификатор операции`
- `Название операции`
- `Название ИСР` (WBS путь)
- `Блок`, `УГПР`
- `Начало`, `Окончание`
- `Плановое количество нетрудовых ресурсов`

## 3) Люди/Техника

Лист: `Люди техника`  
Ключевые колонки:
- `наименование`
- `категория`
- `ед. изм`
- `план/факт`
- датные колонки

## 4) БДР (P&L)

Лист: `БДР`  
- Первая колонка: статья БДР  
- Месяцы: отдельными колонками (`ЯНВАРЬ`, `ФЕВРАЛЬ`, …)  
- Сценарии: определяются по верхним заголовкам (`ПЛАН`, `ФАКТ`).  
- Прогноз игнорируется.

Иерархия статей строится по отступам в первой колонке.

## 5) БДДС (Cash Flow)

Лист: `БДДС`  
Месячные колонки аналогично БДР.  
Направление определяется по знаку суммы (in/out).

## 6) План продаж

Лист: `план продаж`  
- Первая колонка: наименование позиции/объекта  
- Месяцы: отдельными колонками (`ЯНВАРЬ`, `ФЕВРАЛЬ`, …)  
- Сценарии: определяются по верхним заголовкам (`ПЛАН`, `ФАКТ` или `ПРОДАНО`).  
- Прогноз игнорируется.
